# TCP-proxy

# How to start   
• clone repository: 
```
git clone https://github.com/EvtDanya/TCP-proxy
``` 
• go to install folder and install requirements:  
Linux:    
```
chmod +x install.sh
sudo ./install.sh
```   
Windows:  
double click on BAT-file  

# Abilities
To see what this tool can, run the following command:
  ```
  python3 ./tcp-proxy.py -h
  ```
